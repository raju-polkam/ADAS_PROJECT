# Automated Accident Detection and Alert System

## Description
This project detects road accidents using YOLO models and automatically sends alerts using Twilio.

## Technologies Used
- Python
- Flask
- OpenCV
- YOLO
- Twilio
- MySQL

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Configure MySQL database.

3. Update config.py with your credentials.

4. Run:
   python app.py# Automated Accident Detection and Alert System
